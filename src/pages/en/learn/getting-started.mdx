## What is this?

It's a framework to build your own Discord blazingly fast bots on top Cloudflare Workers.

## Key Features

- Free hosting: Your bots can be deployed on a free subscription.
- Serverless: No server has to be maintained.
- Lightweight: The framework is small and really simple to understand.
- Flexible: Even if you are abstracted away from the Discord API, you can still interact with it.
