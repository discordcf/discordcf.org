`ApplicationCommandHandler` is a type that is used to create an application command handler.

```typescript
type ApplicationCommandHandler = (request: Request, ...extra: any) => Promise<any>;
```

## Usage

Used when you call the function `createApplicationCommandHandler`.

```typescript
const handler: ApplicationCommandHandler = applicationCommandHandler = createApplicationCommandHandler(
    application,
    env,
    context,
);
```

### Parameters

- `request`: the Cloudflare request object of type `Request`.
- `extra`: more arguments you may need.

### Return

- A `Promise<any>` as it's an `async` method.
